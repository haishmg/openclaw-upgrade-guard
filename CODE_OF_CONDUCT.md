# Code of Conduct

Clawback follows the Contributor Covenant Code of Conduct, version 2.1.

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and direct.
- Assume contributors are trying to make upgrades safer.
- Keep discussions technical and actionable.
- Avoid sharing private reports, credentials, phone numbers, or local workspace details.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Publishing private user data or secrets.
- Repeatedly derailing issue or pull request discussions.

## Enforcement

Maintainers may edit, hide, or remove comments and may restrict participation for behavior that violates this code.
